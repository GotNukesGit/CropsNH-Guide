Guide corrections
GUIDE
Trampling: Resistance 31 only blocks trampling from running entities. Falling trample ignores Resistance at all values — only fall distance matters. Use trample-immune soil (Garden Soil, Enchanted Earth, Fertilized Dirt) for full protection.
GUIDE
Weed spreading: Resistance ≥ Growth does NOT alone stop outward weed spreading. Both Resistance ≥ Growth AND WeedEx stocked are required. Without WeedEx, outward spreading occurs regardless of Resistance.
GUIDE
Spade mechanics clarified: Breaking any crop (left-click, any maturity) returns a seed via resistance check but destroys the crop — net zero. Right-click harvesting a mature crop with the Spade gives produce + res/31 seed chance — the net-positive action. Reinforced Spade uses guaranteed floor(Resistance/10) formula instead.
GUIDE
Weeds do not destroy crops. When a weed spreads toward an occupied crop stick it calls transferDisease() — making the neighbour sick, not replacing it. Crops are only replaced when a weed spawns on an empty stick.
GUIDE
Crop stick recipe: corrected to 1 log + Saw + File → 2 GT Long Sticks; 4 GT Long Sticks → 4 crop sticks. Previously listed as "2 regular sticks."
GUIDE
"Cross Path" renamed to "Spreading" throughout guide and calculator to reduce confusion with IC2 terminology.
GUIDE
"Crop Tick" terminology — all references to "tick" that meant the 256-game-tick crop update cycle now say "crop tick" explicitly to distinguish from game ticks.
GUIDE
Gaia Wart: Correctly describes the conversion mechanic — grow Nether Wart to maturity, then right-click repeatedly with Snow Blocks (not snowballs) until it converts. Not breedable via any other method.
